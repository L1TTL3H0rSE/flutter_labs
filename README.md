# 🛡️ Security Labs (Лабораторные работы по ИБ)

[![Download APK](https://img.shields.io/badge/Download-APK-green.svg?logo=android)](https://github.com/L1TTL3H0rSE/flutter_labs/releases/latest)

Мобильное приложение на **Flutter**, представляющее собой сборник лабораторных работ по дисциплине, связанной с информационной безопасностью и защитой данных в мобильных приложениях.

Приложение спроектировано в виде единого хаба (Dashboard), откуда можно перейти к выполнению конкретной лабораторной работы. 

## 📋 Список реализованных работ

* **Лабораторная работа 1 и 2: Шифры**
  Реализация классических алгоритмов шифрования: **Атбаш** и **Шифр Цезаря**. 
  Поддерживается шифрование и расшифровка текста на русском и английском языках с сохранением регистра.

* **Лабораторная работа 3: OAuth 2.0**
  Реализация авторизации через сторонний сервис (ВКонтакте). Использование `WebView` для получения `access_token` по протоколу OAuth 2.0.

* **Лабораторная работа 4: Биометрия и ПИН-код**
  Защита доступа к "секретному разделу" приложения с помощью нативных средств устройства (отпечаток пальца, Face ID или системный ПИН-код).

* **Лабораторная работа 5 и 6: Формы, Валидация и Базы данных**
  Реализация безопасной регистрации и авторизации. 
  * Строгая валидация паролей (минимальная длина, наличие цифр и спецсимволов).
  * Локальное хранение данных в **SQLite**.
  * Хеширование паролей перед сохранением в БД с использованием **BCrypt** (соль + хеш).

* **Лабораторная работа 7: Аудит разрешений и Privacy**
  Анализ и управление системными разрешениями (Камера, Микрофон, Геолокация, Биометрия). Отображение текущего статуса доступов и возможность запросить их напрямую из приложения.

## 🛠 Стек технологий и зависимости

В проекте используются следующие популярные пакеты из [pub.dev](https://pub.dev/):

* `flutter/material.dart` — UI фреймворк.
* `local_auth` — для работы с биометрией.
* `sqflite` & `path` — для работы с локальной базой данных SQLite.
* `bcrypt` — для надежного хеширования паролей.
* `permission_handler` — для проверки и запроса системных разрешений.
* `webview_flutter` — для отображения окна авторизации ВКонтакте (OAuth).

## 📁 Структура проекта

Проект разбит на модули для соблюдения принципа единственной ответственности:

```text
lib/
 ┣ main.dart                # Главное меню и навигация (Dashboard)
 ┗ labs/
   ┣ lab1_ciphers.dart      # Логика шифров (Атбаш, Цезарь)
   ┣ lab3_oauth.dart        # Авторизация ВК (WebView)
   ┣ lab4_biometrics.dart   # Экран проверки биометрии
   ┣ lab5_database.dart     # SQLite, хеширование, экраны входа и регистрации
   ┗ lab7_permissions.dart  # Экран управления разрешениями ОС
```

## 🚀 Запуск проекта

### Требования
* Установленный [Flutter SDK](https://docs.flutter.dev/get-started/install).
* Настроенный эмулятор (Android/iOS) или физическое устройство.

### Установка
1. Склонируйте репозиторий:
   ```bash
   git clone <url-вашего-репозитория>
   ```
2. Перейдите в папку с проектом:
   ```bash
   cd security_labs
   ```
3. Установите зависимости:
   ```bash
   flutter pub get
   ```
4. Запустите приложение:
   ```bash
   flutter run
   ```
